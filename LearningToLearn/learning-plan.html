Learning plan (0.5 hours)

↵ Back to module homepage

Carefully read the course outline for this subject.

  1. State the grade you aim to try to get for this subject;
  2. List which assignments you plan to do to receive this grade;
  3. Provide a timeline, showing when you plan to finish each module/assignment.

This learning plan is not a formal contract. You don't need to follow it exactly; later you might change your mind about what grade you want, or what modules you want to do, or how quickly you want to do them. This learning plan is just an opportunity for you to get familiar with the way this class will be run, to think about what you need to do to succeed in this class, and (most importantly) to check your understanding (so that if you have misunderstood something about the grades or the class arrangement, your instructor will have a chance to let you know).

When you finish this activity, you are done with the module (assuming all your work on this and the previous tasks has been satisfactory). If you are interested in leading a discussion on this module, you can go on to see the suggested discussion topics. Otherwise, you can return to the module homepage to review this module, or return to the class homepage to select a different module or assignment to do now.
