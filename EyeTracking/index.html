Eye-tracking (9-10 hours)

↵ Back to class homepage

Module overview

The goal of this module is for you to learn what eye-tracking is and what it can be used for, and to be aware of the kinds of research questions that can be addressed with eye-tracking. This will involve a mixture of hands-on work and literature searching. By the end of the module, hopefully you will be able to do the following things:

  • Explain the use and benefits of eye-tracking for reading research;
  • Calculate and understand different measures of eye movements during reading;
  • Explain the use and benefits of the visual world paradigm;
  • Recognize and explain how the visual world paradigm can be used to test different kinds of psycholinguistic questions.

Module instructions

This module includes seven tasks (but don't worry, some of them are very short!)

Module activities

  1. What is eye-tracking? (1.5 hours) [Complete before class]
  2. Using eye-tracking to study how people read (1 hour) [Complete before class]
  3. Practice calculating reading times (1-2 hours) [Complete before class]
  4. Reflection on eye-tracking for reading (1 hours)
  5. The visual world paradigm (1 hour) 
  6. Brainstorming about visual world (0.5 hours)
  7. Matching (3 hours)
